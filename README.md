# Bill Splitter (Dong Hesab kon) 💰

A web application to track shared expenses and calculate exactly who owes whom.

## 🚀 Features

*   **Add Expenses:** Easily input members and the amount they paid.
*   **Smart Settlement:** Uses a greedy algorithm to calculate the minimum number of transactions to settle debts.
*   **Auto-Save:** Data is saved to `members.json` so you don't lose it.
*   **Clean UI:** Simple interface built with Bootstrap 5.

## 🛠 Built With

*   **Python** (Logic & Algorithms)
*   **Flask** (Web Framework)
*   **HTML & Bootstrap** (Frontend)

## 💻 How to Run


1. **Setup & Run:**
```bash
   pip install flask
   python sharepayment.py
   ```
Open in Browser:Go to http://127.0.0.1:5000
